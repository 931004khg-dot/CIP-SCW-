## 1. 토류판 좌표 순서 문제 해결

### 문제 증상
```
LIST 결과:
  점1: X=353308.6  Y=459172.9  (좌하)
  점2: X=355258.6  Y=459242.9  (우상) ← 대각선 점프!
  점3: X=355258.6  Y=459172.9  (우하)
  점4: X=353308.6  Y=459242.9  (좌상)
  
연결 순서: 1→2→3→4 = 좌하→우상→우하→좌상
결과: X자 교차 (자기교차 폴리곤)
```

### 근본 원인

entmake에서 점 순서가 잘못됨:
- 기존: `pt1 → pt2 → pt3 → pt4`
- 실제 위치 분석:
  - pt1 = pt1 + 25dx + (width/2)perp → 한쪽 끝
  - pt2 = pt2 - 25dx - (width/2)perp → 반대쪽 끝 (대각선!)
  - pt3 = pt2 - 25dx + (width/2)perp → 반대쪽 끝의 다른 면
  - pt4 = pt1 + 25dx - (width/2)perp → 처음 쪽의 다른 면

### 해결 방법

직사각형 순서로 재배열:

```lisp
;; ❌ 잘못된 순서 (교차 발생)
(cons 10 panel-pt1)  ; pt1 쪽, +perp
(cons 10 panel-pt2)  ; pt2 쪽, -perp (대각선 점프!)
(cons 10 panel-pt3)  ; pt2 쪽, +perp
(cons 10 panel-pt4)  ; pt1 쪽, -perp

;; ✅ 올바른 순서 (직사각형)
(cons 10 panel-pt4)  ; pt1 쪽, -perp (한쪽 면 시작)
(cons 10 panel-pt2)  ; pt2 쪽, -perp (같은 면 끝)
(cons 10 panel-pt3)  ; pt2 쪽, +perp (반대 면 시작)
(cons 10 panel-pt1)  ; pt1 쪽, +perp (반대 면 끝)
```

### 예상 결과

```
LIST 결과:
  LWPOLYLINE  도면층: "_토류판(timber)"
    점 순서: pt4 → pt2 → pt3 → pt1 (닫힘)
    
    점4: (pt1 - 25dx - 35perp)
    점2: (pt2 - 25dx - 35perp)
    점3: (pt2 - 25dx + 35perp)
    점1: (pt1 + 25dx + 35perp)
    
  면적: 136,500 mm² (1950mm × 70mm)
  둘레: 4040 mm (2×(1950+70))
  
  ✅ 정상적인 직사각형, 교차 없음
```

## 2. H-Pile 필렛 위치 수정

### 요구사항
웹과 플랜지가 만나는 **정확한 접합부** 4곳에 라운드 코너(필렛) 추가  
**필렛 반지름**: 웹 두께(tw) × 2

### 문제점 (이전 구현)

```lisp
;; ❌ 잘못된 필렛 위치
(cons 10 pt1)
(cons 42 0.4142...)  ; pt1→pt2: 플랜지 외부 모서리 (X)
(cons 10 pt2)
(cons 10 pt3)
(cons 10 pt4)
(cons 42 0.4142...)  ; pt4→pt5: 웹 내부 (X)
...
```

**문제**: 필렛이 플랜지 외부 모서리와 웹 내부에 적용됨 → 접합부가 아님!

### 해결 방법

웹-플랜지 **접합부**에만 필렛 적용:

```lisp
;; ✅ 올바른 필렛 위치
(cons 10 pt1)   ; 좌상단 외부
(cons 10 pt2)   ; 우상단 외부
(cons 10 pt3)   ; 우상단 안쪽 (플랜지 끝)
(cons 42 0.4142135623730951)  ; 필렛 1: pt3→pt4 (우상단 접합부) ✓
(cons 10 pt4)   ; 웹 우상단
(cons 10 pt5)   ; 웹 우하단
(cons 42 0.4142135623730951)  ; 필렛 2: pt5→pt6 (우하단 접합부) ✓
(cons 10 pt6)   ; 우하단 안쪽 (플랜지 시작)
(cons 10 pt7)   ; 우하단 외부
(cons 10 pt8)   ; 좌하단 외부
(cons 10 pt9)   ; 좌하단 안쪽 (플랜지 끝)
(cons 42 0.4142135623730951)  ; 필렛 3: pt9→pt10 (좌하단 접합부) ✓
(cons 10 pt10)  ; 웹 좌하단
(cons 10 pt11)  ; 웹 좌상단
(cons 42 0.4142135623730951)  ; 필렛 4: pt11→pt12 (좌상단 접합부) ✓
(cons 10 pt12)  ; 좌상단 안쪽 (플랜지 시작)
```

### 필렛 위치 상세

```
접합부 위치 (그림의 빨간 원 부분):

      pt1 ────── pt2         ← 플랜지 상단 외부
       │          │
    pt12│          │pt3      ← 플랜지 내부
       │          │
       ◯────────◯          ← 필렛 1 (pt3→pt4)
       │          │             필렛 4 (pt11→pt12)
    pt11│   웹    │pt4
       │  (web)  │
       │          │
    pt10│          │pt5
       │          │
       ◯────────◯          ← 필렛 3 (pt9→pt10)
       │          │             필렛 2 (pt5→pt6)
     pt9│          │pt6      ← 플랜지 내부
       │          │
      pt8 ────── pt7         ← 플랜지 하단 외부
```

**필렛 반지름 계산**:
```lisp
(setq fillet-r (* tw 2.0))  ; H 298×201×9/14 → 18mm
```

**Bulge 값**: 90도 호 = `tan(45°) = 0.4142135623730951`

### 예상 결과

       │                │
     pt9│                │pt6
       │                │
      pt8 <──(필렛3)── pt7
```

## 기술적 세부 사항

### Bulge 값 설명

LWPOLYLINE의 bulge 값은 정점 사이의 호를 정의합니다:

```
bulge = tan(θ/4)

여기서:
- θ = 호의 각도 (라디안)
- 90도 = π/2 라디안
- bulge = tan((π/2)/4) = tan(π/8) = tan(22.5°) ≈ 0.4142135623730951
```

### 방향 벡터 계산

토류판의 직사각형 좌표 계산:

```lisp
;; 방향 벡터 (pt1 → pt2)
dx = (pt2.x - pt1.x) / length
dy = (pt2.y - pt1.y) / length

;; 수직 벡터 (반시계방향 90도 회전)
perp-dx = -dy
perp-dy = dx

;; 토류판 좌표
panel-pt1 = pt1 + 25*direction + (width/2)*perpendicular
panel-pt2 = pt2 - 25*direction - (width/2)*perpendicular
panel-pt3 = pt2 - 25*direction + (width/2)*perpendicular
panel-pt4 = pt1 + 25*direction - (width/2)*perpendicular
```
